Hello, World

19 Jan, 2025

This is going to be the first post published on this new iteration of my website. Fun fact, I had initially thought I’d be using Pika for my new homepage, and I’d recommend checking it out if you want a no-friction blogging tool! However, if I was going to start writing more, I wanted to do so with complete control of where my content was hosted and how it was rendered– which meant going all-in on using Hugo (with the lovely Hugo ʕ•ᴥ•ʔ Bear Blog theme, which is inspired by another minimalist indie blogging platform, Bear Blog).

I am now hosting this site on GitHub Pages, but I can easily move it somewhere else as it is a nice self-contained bundle. This also gives me peace of mind that I can add content like portfolio work in here and don’t need to worry about it going down. It might not be the best presentation for certain types of media, and I’ll be figuring that out as I add more to this site, but what matters most to me is keeping all that work easily accessible from one stable place.

I think that one post I made on Pika did outline my feelings and motivation quite well, so I’m reproducing it here.
